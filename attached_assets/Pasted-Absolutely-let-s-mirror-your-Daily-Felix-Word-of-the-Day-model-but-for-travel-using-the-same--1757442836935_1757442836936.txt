✨ What It Is

Daily Felix – City of the Day is a lightweight, AI-powered travel companion web app that introduces users to a new city every day — with curated cultural insights, local flavor, fun facts, and practical travel tips.

It’s built to spark wanderlust, educate global citizens, and guide future trips — one city, one day at a time.

⸻

🔑 Key Features (Mirrored to Word of Day)

🧭 1. Daily Travel Content (AI-Powered + Custom)
	•	Each day, users receive a featured city with:
	•	✨ Landmark Highlight
	•	🍜 Local Food or Drink
	•	💸 Budget/Insider Travel Tip
	•	🤓 Fun Fact or Cultural Insight
	•	Optional: Embed affiliate buttons (e.g., “Book a Hotel”, “Explore Tours”, etc.)

⸻

📱 2. User Dashboard – Interactive Card System
	•	Three key content cards per day:
	1.	Morning: “Wake up in [City]” + Landmark/Intro
	2.	Afternoon: Food & Culture
	3.	Evening: Budget Tip + Travel Quote or Fun Fact
	•	Bonus card: “Wander Wisdom” – travel-related life reflections

⸻

🛠️ 3. Admin Panel (You)
	•	Auto-generate daily cities using AI prompts
	•	Approve/edit content before publishing (or let it run on full autopilot)
	•	Add your own notes, travel stories, or affiliate links per city
	•	Pin special cities (e.g., holidays, event locations, promos)

⸻

🔔 4. Smart Notifications (Optional, Configurable)
	•	3–5 reminders daily (light, not overwhelming like 18)
	•	Morning: Today’s city
	•	Afternoon: Food/culture prompt
	•	Evening: Travel tip/fact

⸻

📲 5. PWA-Ready (Installable Web App)
	•	Works offline after caching
	•	Users can install on mobile or desktop
	•	Collect “City Postcards” as they visit more cities

⸻

🔐 6. User Authentication
	•	Secure login with email or social
	•	“Stay signed in” toggle
	•	Optional profiles: track favorite cities, create a “bucket list,” etc.
