📦 Replit Developer Instructions – City of the Day™ Web App (MVP Build)











🧭 OVERVIEW:





City of the Day™ is a lightweight, AI-powered travel inspiration web app that shows users a new global city each day — with curated highlights, local food tips, budget hacks, and a fun fact.



This is not a travel planner or booking site. It is a daily travel companion, offering micro-learning and wanderlust for everyday users.



The MVP should be clean, mobile-first, installable as a PWA, and built for daily usage with a clear upgrade path to a paid version.









🔑 CORE MVP FEATURES TO BUILD







1. 🔐 

User Login System (Authentication)





Email + password or magic link sign-in
“Keep me signed in” toggle
Guest access allowed (view-only mode)










2. 📅 

Daily City Generator (Content Engine)





Show one new city per day to all users
Admin can:
Preload or pin city for any date
OR let AI/randomizer choose next city from list

Use a static JSON or lightweight CMS (Notion, Airtable, etc.) to store city data
For now, pre-load at least 7 demo cities (e.g., Lisbon, Cape Town, Tokyo, Buenos Aires, San Francisco, Marrakech, Bangkok)










3. 📲 

City Display Card Stack (Frontend UI)





Each city card must include:

Card

Content

🌍 City Intro

City name, country, cover image, 1-paragraph description

🏛️ Landmark

1 sentence about a top landmark (e.g., Eiffel Tower in Paris)

🍜 Local Food

Name + short food/drink tip

💸 Budget Tip

Money-saving travel insight

🤓 Fun Fact

Quirky cultural or historical insight

🔗 CTA Buttons

“Book Hotel”, “Explore Tours”, “Search Flights” (use placeholders for now)

All cards shown together or in scroll/swipe format (user can browse in one motion).









4. 🧳 

City Log / “Postcard” Tracker (Free User Dashboard)





Logged-in users can:
View today’s city
Save it to their collection (favorite/bookmark)
See a list of all previously saved cities (but NOT access full content unless they’re premium)











5. 💳 

Premium Tier Logic (Paywall System)





Free Tier:
Only today’s city visible
No access to past cities
Limited travel tips

Premium Tier:
Access to previous cities (archive view)
Downloadable city guide (PDF)
Custom affiliate links (coming later)
Weekly “Theme Drops” (e.g., “7 Coastal Cities”)





Use Stripe or Gumroad for payments, with monthly ($7) or yearly ($49) subscription options.









6. 📱 

PWA-Ready





App must be installable on mobile & desktop
Works offline for cached cities
Display simple splash screen when loading offline










7. 🧑‍💻 

Admin Panel (Basic)





Admin can:
Add/edit cities manually
Upload city images
Generate a new city using AI prompt:
"Create a 100-word travel guide for [City Name] including a famous landmark, local food or drink, a budget travel tip, and a fun cultural fact."
Pin featured cities
Track usage (basic analytics like city opens, saves)











8. 🌐 

Responsive Design





Must be mobile-first
Use clean typography, travel-inspired visuals, soft card transitions
Design inspiration: Calm App, Duolingo, Notion, or Blinkist
